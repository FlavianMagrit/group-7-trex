Bienvenue dans le README de notre adaptation du célèbre Goomba's Revenge: Mario's Challenge. Ce projet a été réalisé dans le cadre de notre WorkShop EFFICOM.

## Description du jeu

Goomba's Revenge: Mario's Challenge est un jeu de plateforme en ligne dans lequel vous incarnez le célèbre plombier Mario. 
Dans ce monde, vous devez éviter les ennemis habituels de Mario, tels que les Goombas et les Koopa Troopas, tout en récupérant des champignons et peachs pour obtenir des avantages et des bonus.
Votre objectif est d’atteindre le drapeau de fin, tout en évitant ces ennemis.
En conclusion Goomba's Revenge: Mario's Challenge est un jeu de plateforme dans lequel votre seul objectif est d’obtenir les meilleurs score donc Préparez-vous à affronter des défis et à découvrir ce nouveau monde dans ce jeu en ligne divertissant.
## Fonctionnalités

- Tableau des scores pour suivre votre progression
- Connexion / inscription
- Obtention de plusieur vie et bonus de score
- Mode plein écran pour une immersion totale

# Cloner le projet
## Prérequis

Vous devez disposer d'un navigateur web moderne (Chrome, Firefox, Safari ou Edge) et d'une connexion Internet.

## Installation

1. ```nvm use 16.20.0```
2. ```npm install```
3. ```npm run dev```
4. open ```localhost:8080```


## Comment jouer

1. Appuyez sur la barre d'espace pour faire sauter Mario.
2. Collectez des champignon pour augmenter votre vie.
3. Attrapez Peach pour augmenter votre score.
3. Essayez de battre votre meilleur score et celui des autres joueurs !

## Contrôles

- Barre d'espace : Sauter
